# Sliding Puzzle

Simple 4×4 sliding puzzle with support for multiple players.

How to use
- Open [sliding-puzzle/puzzle.html](sliding-puzzle/puzzle.html) in a browser.
- Open [sliding-puzzle/puzzle.html](sliding-puzzle/puzzle.html) in a browser.
- Choose number of players (1–4) — the page updates automatically when you change the number.
- Click "Shuffle" to set a single puzzle for all players. Timers start when a player makes their first move.
- Tiles slide into the blank spot; complete the board 1→15 to win.
- Each player tracks moves and time independently.

Notes:
- The UI adapts to light/dark desktop themes via `prefers-color-scheme`.
- Per-player controls: Pause, Resume, Restart, Reset are available in each player panel.
